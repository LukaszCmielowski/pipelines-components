
## AutoRAG specific components. An utility package  

This package is a collection of _components_, _images_, _scripts_ and other various enhancements built upon the `kfp` client library.  
Its main purpose is to faciliate the creation of AutoRAG pipelines and lay a strong foundation for further community-driven advancements.  
The building blocks included here provide the fastest way to start **creating and enhancing your own AutoRAG pipelines**.
