# Components Components

This directory contains components of the **automl** organization in the **Components** category:

- [Tabular Data Loader](./tabular_data_loader/README.md): Loads a tabular CSV dataset from an S3-compatible bucket using AWS credentials.
- [Train Test Split](./train_test_split/README.md): Splits a tabular dataset into train and test sets and writes them to output artifacts.
