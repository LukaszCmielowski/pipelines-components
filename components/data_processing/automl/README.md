# Components Components

This directory contains components of the **automl** organization in the **Components** category:

- [Tabular Data Loader](./tabular_data_loader/README.md): Downloads a dataset from S3, samples 50% of the rows, and saves the sample to the output artifact.
- [Tabular Train Test Split](./tabular_train_test_split/README.md): Splits a tabular dataset into train and test sets and writes them to output artifacts.
