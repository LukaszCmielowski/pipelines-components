# Data Processing Components

This directory contains components in the **Data Processing** category:

- [Tabular Data Loader](./automl/tabular_data_loader/README.md): Downloads a dataset from S3, samples 50% of the rows, and saves the sample to the output artifact.
- [Tabular Train Test Split](./automl/tabular_train_test_split/README.md): Splits a tabular dataset into train and test sets and writes them to output artifacts.
- [Yoda Data Processor](./default/yoda_data_processor/README.md): Prepare the training and evaluation datasets by downloading and preprocessing.
